# Зарплатный калькулятор

Программа предназначена для расчёта заработной платы сотрудника с учётом оклада, премии, районного коэффициента, количества рабочих и отработанных дней, а также НДФЛ.
Разработана, для лабораторной работе по Java.

## Используемые технологии

* Java
* Swing
* Visual Studio Code

## Файлы проекта

* `SalaryCalculatorGUI.java` — исходный код программы
* `SalaryCalculatorGUI.class` — скомпилированный файл программы
* `start.bat` — файл для быстрого запуска программы

## Первый запуск через javac

Перед запуском необходимо убедиться, что установлен JDK.

Проверка Java:

```bash
java -version
```

Проверка компилятора:

```bash
javac -version
```

Для компиляции программы необходимо открыть терминал в папке с файлом `SalaryCalculatorGUI.java` и выполнить команду:

```bash
javac SalaryCalculatorGUI.java
```

После успешной компиляции появится файл:

```text
SalaryCalculatorGUI.class
```

Запуск программы через командную строку:

```bash
java SalaryCalculatorGUI
```

## Запуск через bat-файл

Для удобного запуска программы используется файл `start.bat`.

Содержимое файла:

```bat
@echo off
java SalaryCalculatorGUI
pause
```

Для запуска программы достаточно дважды нажать на файл `start.bat`.

## Работа с программой

1. Запустить программу.
2. Ввести оклад сотрудника.
3. Ввести процент премии.
4. Ввести районный коэффициент.
5. Ввести количество рабочих дней.
6. Ввести количество отработанных дней.
7. Нажать кнопку «Рассчитать».
8. Получить итоговую сумму к выплате.

## Возможные ошибки

Если программа не запускается, необходимо проверить:

* установлена ли Java;
* доступна ли команда `java` в командной строке;
* находится ли файл `SalaryCalculatorGUI.class` рядом с `start.bat`;
* совпадает ли имя файла с именем класса `SalaryCalculatorGUI`.
